# dotfiles

![.dot](dot.png)

## Description

My dotfiles for a linux distribution configuration.

**WM**: `i3`

**Shell**: `zsh` (`+ohmyzsh`, `+ohmyposh`)

**Terminal**: `alacritty`

**Editor**: `vim` / `vscode`

**File manager**: `thunar`

**Bar**: `polybar`

**Compositor**: `compton`

**Notifications**: `dunst`

**App launcher**: `rofi`

**Searcher**: `ripgrep`

**File finder**: `fd`

**Fuzzy finder**: `fzf`

**File listing**: `exa`

**Git CLI**: `tig`

**Font**: `Iosevka` *(custom)*
